# Emotion Recognition Using FER-2013 Dataset

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) to classify facial expressions into different emotions using the FER-2013 dataset.

## 📷 Categories:
- Angry
- Disgust
- Fear
- Happy
- Sad
- Surprise
- Neutral

## 🛠️ Technologies Used:
- Python
- TensorFlow / Keras
- Google Colab
- Matplotlib, Pandas, NumPy

## 📈 Model Performance:
Achieved an accuracy of **57%** on test data.

## 📂 How to Run:
- Open the Colab notebook
- Upload the FER-2013 dataset
- Run all the cells

## 🤖 Future Work:
- Improve accuracy with data augmentation
- Try different CNN architectures
